# Banner 组件

为了统一技术架构平台的风格，当然不会放过 Banner 的自定义

```text
                         _ _

|_| |_| |_|\___\___/ \__,_|_|_| |_|\__, |
                                   |___/
Spring Boot Version: 2.6.3
```

## 使用方法

在工程的 Start 模块添加依赖

```xml
<!-- mcoding自定义starter -->
<dependency>
    <groupId>cn.mcoding</groupId>
    <artifactId>mcoding-spring-boot-starter-banner</artifactId>
</dependency>
```
